Help and compatibility

Support

Most issues can be resolved by opening the exact Codex Analytics usage route and reloading once. If the page layout has changed, the popup will report the latest compatibility state.

Open a support issue

Quick checks

  1. Open https://chatgpt.com/codex/cloud/settings/analytics and select the Usage view.
  2. Confirm Chrome is version 105 or newer and Burn Rate Coach is enabled.
  3. Reload Analytics. The extension runs only on that exact usage route.
  4. Open the extension popup and check Codex compatibility.
  5. If a card remains native, wait for the page to finish loading and reload once. Burn Rate Coach deliberately restores native cards after a failed 900 ms enhancement attempt.

What to include

  • Chrome and extension versions;
  • whether you entered Analytics directly or through Codex navigation;
  • light or dark theme, viewport width, and browser zoom;
  • the popup’s compatibility message;
  • whether reloading changes the result.
Protect your account.

Never post cookies, authentication tokens, credentials, prompts, conversations, full page exports, private source code, or screenshots containing account data. Synthetic or tightly cropped UI evidence is preferred.

Known limitations

  • History is collected only while Analytics is open.
  • Closed-page activity appears as a gap between observations.
  • The current parser and interface are English-language only.
  • Codex interface changes can temporarily prevent card recognition.
  • An unpacked install and Chrome Web Store install use different extension storage identities.

Private contact

For a privacy question or a report that should not be public, email burn.rate.coach@gmail.com. Do not email credentials or account exports.
