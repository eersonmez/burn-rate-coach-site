ملخص سياسة الخصوصية

Burn Rate Coach إضافة مستقلة لمتصفح Chrome تضيف إرشادات لوتيرة الاستخدام إلى صفحة استخدام Codex Analytics. وهي غير تابعة لـ OpenAI أو معتمدة منها.

ترجمة أولية

يحتاج هذا الملخص إلى مراجعة بشرية من متحدث متمكن. النص الرسمي هو السياسة الإنجليزية: سياسة الخصوصية بالإنجليزية

المعلومات التي تتم معالجتها

أثناء فتح Codex Analytics، يراقب جسر صفحة مضمن بصورة سلبية نتيجة الحصة الناجحة الخاصة بالصفحة عندما تفك الصفحة ترميزها. ولا يصل مباشرة إلا إلى معرّفات مقياس وفترة حد المعدل الثابتة والنسبة المستخدمة ومدة فترة الحصة وتوقيت إعادة التعيين اللازمة للإرشاد. تربط الإضافة المعزولة النتائج بالبطاقات الأصلية فقط عبر بيانات التقدم وإعادة التعيين الرقمية المتطابقة؛ ولا تستخدم أبدًا التسميات المترجمة أو ترتيب البطاقات. تنشئ الإضافة طابع وقت الرصد محليًا بدلًا من قراءته من الاستجابة. لا يبدأ الجسر الطلب، ولا ينسخ الاستجابة أو يفك ترميزها بصورة مستقلة، ولا يفحص الترويسات أو ملفات تعريف الارتباط أو رموز التفويض أو بيانات الاعتماد الأخرى.

قد تتضمن نتيجة الصفحة أيضًا البريد الإلكتروني ومعرّفات الحساب والمستخدم ومعلومات الخطة والأرصدة وحقول التحكم في الإنفاق. لا يصل Burn Rate Coach إلى هذه الحقول غير ذات الصلة ولا يسردها، ولا يمررها مطلقًا عبر الجسر أو يسجلها أو يخزنها أو يرسلها. كما لا يقرأ أو يخزن المطالبات أو المحادثات أو الشفرة المصدرية أو كلمات المرور أو بيانات اعتماد الدفع أو مفاتيح API.

يخزن Burn Rate Coach الإعدادات التي تختارها. عند تفعيل السجل المحلي، تُحفظ نسب الحصة ذات الطابع الزمني وأوقات إعادة التعيين في التخزين المحلي للإضافة في Chrome. تُحذف السجلات الأقدم من 14 يومًا عند معالجة Analytics للسجل في المرة التالية، ويقتصر كل مقياس على 10,082 عينة.

كيفية استخدام المعلومات

تُستخدم معلومات الحصة المسموح بها فقط لحساب وعرض موضع الوتيرة الحالي والإجراء الموصى به والمسارات المحلية والحصة المتوقعة عند إعادة التعيين وأحدث حالة توافق نشطة أو غير معروفة.

التخزين والإرسال

تبقى الإعدادات وحالة التوافق وسجل الحصة في chrome.storage.local على جهازك. لا يملك Burn Rate Coach خدمة خلفية، ولا يرسل بيانات الإضافة إلى المطور أو أطراف ثالثة، ولا يستخدم حزم تحليل أو إعلانات، ولا يبيع بيانات المستخدم.

إذا اخترت رابطًا خارجيًا للدعم أو الخصوصية، يفتح Chrome ذلك الموقع العام. ولا يضيف Burn Rate Coach معلمات تتبع إلى الرابط.

المشاركة

لا يشارك Burn Rate Coach بيانات الإضافة مع أطراف ثالثة ولا ينقلها إليها. ولا يستخدم البيانات للإعلانات أو إنشاء الملفات التعريفية أو تقييم الجدارة الائتمانية أو لأي أغراض لا تتعلق بوتيرة الحصة.

عناصر التحكم المتاحة لك

تشرح صفحة Chrome Web Store هذا الاستخدام المحلي لبيانات الحصة قبل التثبيت، وتكرره النافذة المنبثقة بلغة بسيطة. يؤدي إيقاف إظهار المرشد إلى إزالة الإرشادات، لكنه لا يوقف مراقبة الصفحة أو فحوص التوافق. ويؤدي إيقاف سجل الوتيرة المحلي إلى وقف جمع العينات المحفوظة واستخدامها. يحذف مسح السجل لقطات الحصة الحالية والقديمة. عند إلغاء تثبيت Burn Rate Coach، يزيل Chrome بياناته المحلية.

الأمان والامتثال للسياسة

تأتي كل الشفرات القابلة للتنفيذ مضمنة مع الإضافة. يرسل جسر الصفحة إلى سياق الإضافة المعزول إسقاطًا مسموحًا به فقط لبيانات الحصة، ولا يرسل الاستجابة الخام. يستخدم Burn Rate Coach المعلومات لغرضه الوحيد المعلن ويلتزم بمتطلبات الاستخدام المحدود في سياسة بيانات مستخدمي Chrome Web Store.

التغييرات

ستنعكس التغييرات الجوهرية في معالجة البيانات في هذه السياسة وإفصاحات الخصوصية في Chrome Web Store، وعند اللزوم، في إفصاح داخل المنتج قبل بدء المعالجة الجديدة.
